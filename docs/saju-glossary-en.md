# Saju English Glossary And Label System

This glossary is intended for an English-language Saju product. The UI should keep technical terms available, but explain them in plain language.

## Consumer Copy Rules

- Show Korean/Hanja terms only after the plain-English label.
- Explain every technical marker in one sentence first.
- Never present a star as a curse.
- Translate harsh terms into coaching language.
- Use `can`, `may`, `tends to`, and `often` instead of deterministic wording.

Example:

> Needle Star means your words and observations can be unusually precise. At its best, this is surgical insight; under stress, it can become a sentence that lands harder than you intended.
